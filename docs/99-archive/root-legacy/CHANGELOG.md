## Migration Guide

### From 3.x to 4.0

1. **Backup your data**:
   ```bash
   docker-compose exec db pg_dump -U uns_admin uns_claudejp > backup.sql
   ```

2. **Stop old services**:
   ```bash
   STOP.bat
   ```

3. **Pull latest code**:
   ```bash
   git pull origin main
   ```

4. **Start new version**:
   ```bash
   START.bat
   ```

5. **Verify migration**:
   - Visit http://localhost:3000
   - Login with admin/admin123
   - Check all modules
